# Fashion Boutique Inventory Analysis using Python

# Project Overview

## Objectives
- Read and analyze fashion boutique data from a CSV file
- Understand stock distribution across categories and brands
- Analyze discounts and pricing strategies
- Calculate return rate and customer satisfaction
- Visualize insights using charts
- Practice structured and modular Python programming

## Dataset Description

The dataset ("fashion_boutique_dataset.csv") contains the following columns:

product_id – Unique product identifier  
category – Product category (Men, Women, Kids, etc.)  
brand – Brand name  
season – Season of the product  
size – Product size  
color – Product color  
original_price – Original product price  
markdown_percentage – Discount percentage  
current_price – Discounted price  
purchase_date – Date of purchase  
stock_quantity – Available stock  
customer_rating – Customer rating (1–5)  
is_returned – Whether the product was returned  
return_reason – Reason for return (if any)
